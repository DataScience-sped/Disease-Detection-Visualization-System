# 🌍 Disease Detection & Visualization System

## 📌 Overview
This project is a web-based disease visualization system that displays different diseases across various geographical locations on a world map. It uses color-coded markers and circle sizes to represent disease type and severity (number of affected cases).

The system combines **Python (Folium)** for map generation and **HTML/CSS** for frontend visualization.

## 🎯 Features
* 🌎 Interactive world map using Folium
* 🏥 Displays hospitals and nursing homes with markers
* 🎨 Color-coded diseases for easy identification
* 📊 Circle size represents number of affected cases
* 📍 Popups showing disease name and affected count
* 📌 Custom legend for disease classification
*

## ⚙️ How It Works
1. The Python script (`app.py`) creates datasets for:
   * Locations (Hospitals/Nursing Homes)
   * Diseases with affected counts
2. Using **Folium**:
   * Markers are added for hospitals and nursing homes
   * Circle markers represent diseases
   * Colors indicate disease type
   * Radius indicates severity (affected people)
3. A custom legend is added using HTML.
4. The final map is saved as:
   👉 `disease_analysis_map.html`
5. The `index.html` acts as the homepage and links to the map.

## 🚀 How to Run the Project
1. Install required libraries:
   ```bash
   pip install folium pandas
   ```
2. Run the Python file:
   ```bash
   python app.py
   ```
3. Open:
   * `index.html` → Home page
   * Click the button to view the map
